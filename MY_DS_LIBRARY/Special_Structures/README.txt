How to Print Console Output in a file?

1) open terminal (a.k.a cmd interface) & change dir to same path of your .exe (provided it's present)
  [step 1: type d: or D: to change parent directory (if needed)
   step 2: copy path of your targeted folder and after typing: cd {space} {paste path}]

2) type command: {filename.exe} Optimized_Multi_Attribute_AVL_Tree_Adapter.exe > out.txt

Note: project {headers} files are directory dependant.
